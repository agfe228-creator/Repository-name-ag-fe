주요 카테고리

HTML

문서 구조

태그의 역할, 링크, 이미지, 폼처럼 웹 문서의 기본 뼈대를 다룹니다.

CSS

스타일과 레이아웃

여백, 정렬, flex, position, 반응형처럼 화면을 보기 좋게 만드는 방법을 정리합니다.

JavaScript

동작과 데이터

변수, 함수, 배열, 이벤트, fetch처럼 화면에 움직임을 더하는 기초를 설명합니다.

React

컴포넌트 입문

컴포넌트, props, state, Hook, 목록 렌더링처럼 React 시작에 필요한 내용을 다룹니다.

Git/Web

배포와 웹 기초

Git, GitHub, 도메인, HTTPS, 정적 사이트 배포 흐름을 쉽게 정리합니다.
