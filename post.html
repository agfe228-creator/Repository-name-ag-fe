프론트엔드 기초 학습 글

AG FE Guide는 HTML, CSS, JavaScript, React, Git을 처음 배우는 사람을 위해 작은 예제와 확인 순서 중심으로 글을 제공합니다.

이 페이지에서 다루는 내용

각 글은 하나의 개념이나 문제 상황을 기준으로 구성됩니다. 먼저 어떤 상황에서 필요한 개념인지 설명하고, 짧은 코드 예제와 함께 브라우저에서 확인할 부분을 정리합니다.

JavaScript가 실행되는 환경에서는 주소의 글 식별자에 맞는 상세 글이 자동으로 표시됩니다. 스크립트를 읽지 못하는 환경에서도 사이트 주제와 주요 학습 경로를 확인할 수 있도록 기본 안내를 함께 제공합니다.

대표 학습 경로

  • HTML 문서 구조와 태그 의미 이해하기
  • CSS 선택자, 여백, 정렬, 반응형 기초 확인하기
  • JavaScript 변수, 함수, 배열, 이벤트 흐름 익히기
  • React 컴포넌트, props, state 기본 흐름 정리하기
  • Git, GitHub, 배포와 웹 기본 개념 이해하기

작성 기준

AG FE Guide 편집팀은 초보자가 실제 학습 중 자주 만나는 오류와 헷갈리는 용어를 기준으로 글을 구성합니다. 필요한 경우 MDN Web Docs, React 공식 문서, Git/GitHub 공식 문서 등 원문 자료를 함께 안내합니다.
