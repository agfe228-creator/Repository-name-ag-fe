AG FE Guide 소개

AG FE Guide는 프론트엔드를 처음 배우는 사람이 HTML, CSS, JavaScript, React, Git의 기본 개념을 차분히 익힐 수 있도록 정리하는 입문 가이드 사이트입니다.

문법을 한꺼번에 외우게 하기보다 작은 예제, 흔한 실수, 확인 순서, 공식 문서 링크를 함께 제시해 실제 학습 중 막히는 지점을 줄이는 것을 목표로 합니다.

다루는 주제

  • HTML 문서 구조, 링크, 이미지, 폼 기초
  • CSS 선택자, 여백, 정렬, 반응형, 레이어 문제
  • JavaScript 변수, 함수, 배열, 이벤트, fetch 기초
  • React 컴포넌트, props, state, Hook 입문
  • Git, GitHub, 정적 사이트 배포와 웹 기본 흐름
