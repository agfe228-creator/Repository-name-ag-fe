POLICY

운영정책

AG FE Guide는 프론트엔드 입문자가 웹 개발 기초를 이해할 수 있도록 HTML, CSS, JavaScript, React, Git 관련 정보를 정리하는 학습 가이드 사이트입니다.

콘텐츠 이용 안내

사이트에 게시된 글은 일반적인 학습 정보 제공을 목적으로 합니다. 예제 코드는 초보자 이해를 위해 단순화되어 있으며, 실제 프로젝트 환경에서는 브라우저 버전, 프레임워크 설정, 배포 방식에 따라 결과가 달라질 수 있습니다.

작성 기준

글은 초보자가 자주 헷갈리는 개념, 실제로 발생하기 쉬운 오류, 작은 코드 예제를 중심으로 작성합니다. 필요한 경우 MDN Web Docs, React 공식 문서, Git/GitHub 공식 문서 등 원문 자료를 함께 안내합니다.

정정 요청

잘못된 설명, 동작하지 않는 예제, 오래된 공식 문서 링크가 있다면 issuebara@gmail.com 으로 알려주세요. 확인 후 필요한 경우 내용을 수정하거나 보완하겠습니다.

광고와 제휴

향후 사이트에는 광고가 표시될 수 있습니다. 광고 영역은 콘텐츠와 구분되도록 운영하며, 광고 클릭을 유도하는 표현을 사용하지 않습니다.

무단 복제 금지

AG FE Guide의 콘텐츠를 허가 없이 복제하거나 재배포하는 행위를 금지합니다. 코드 예제는 학습 목적으로 참고할 수 있으나, 글 전체를 그대로 옮기는 행위는 허용하지 않습니다.
